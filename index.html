Welcome to my Semantic Web Application

This application is built by Thomas Kamps as an assignment for the Semantic Web course taught at the VU. The main goal of this application is to provide the user with information on a variety of pizzas. Unfortunately the language of the used ontology are in Portugese, but the message is usually understandable. The used ontology is an example ontology that contains all constructs required for the various versions of the Pizza Tutorial run by Manchester University (see http://www.co-ode.org/resources/tutorials/). After you have selected a pizza you will see the used toppings, and get the choice for showing the categories it belongs to with or without inferencing on the ontology. The different results you get can be explained by the fact that in the ontology categories are defined classes and so contain sufficient conditions. So a reasoner will put pizzas that meet those conditions under the defined class. Select your pizza of choice and see the magic:
